# joelinwood.com

Personal site for Joel Inwood — a web and public relations professional working
at the intersection of communications and technology.

A single, self-contained static page styled after an early-'90s NES dialogue box
(green-on-black, Press Start 2P + VT323), served via GitHub Pages at
<https://www.joelinwood.com>.

## Layout

- `index.html` — the entire site (HTML + inline CSS)
- `fonts/` — self-hosted web fonts, so the page makes no third-party requests
- `CNAME` — custom domain configuration
